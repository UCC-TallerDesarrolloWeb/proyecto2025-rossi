## Descripción

**Burger House** es el sitio web de una hamburguesería, desarrollado con HTML, CSS y JavaScript puro. Se puede abrir directamente desde el archivo `index.html`.

## Páginas

| Página | Archivo | Descripción |
|---|---|---|
| Home | `index.html` | Presentación y especialidades destacadas |
| Menú | `menu.html` | Catálogo y configuración de hamburguesas |
| Sucursales | `sucursales.html` | Locales y direcciones |
| Contacto | `contacto.html` | Formulario de consultas |
| Carrito | `carrito.html` | Productos seleccionados y total |

## Tecnologías

- HTML5
- CSS3
- JavaScript
- LocalStorage
- Google Fonts

## Estructura

### Archivos principales

Los cinco documentos HTML están en la raíz. Los estilos están en `css/estilos.css`, la lógica en `js/funciones.js` y los recursos visuales en `imagenes/`.

## Funcionalidades

- Menú con selección de uno, dos o tres medallones.
- Cálculo de precio con papas y bebida opcionales.
- Carrito persistente mediante LocalStorage.
- Eliminación individual y vaciado completo del carrito.
- Formulario de contacto con validación de datos.
