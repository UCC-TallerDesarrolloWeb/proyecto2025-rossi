# Burger House - Proyecto Final DW

## Descripción
Este proyecto consiste en una aplicación web desarrollada con **React** para una hamburguesería. Permite a los usuarios navegar por el menú, personalizar sus pedidos (medallones extra, combos con papas y bebida), gestionar un carrito de compras y contactar al local.

## Autor
* **Nombre:** Lorenzo Rossi Bossio
* **Materia:** Taller de Desarrollo Web
* **Año:** 2025

## Tecnologías Utilizadas
* HTML5 Estructura y contenido semántico
* JavaScript Interactividad y funcionalidades dinámicas
* Git Control de versiones
* React 18 Vite
* Sass (SCSS) Estilado profesional con variables y mixins

## Características Principales
* **Menú Dinámico:** Listado de productos renderizado desde archivos de utilidad.
* **Personalización:** Cálculo de precio en tiempo real según los ingredientes seleccionados.
* **Carrito de Compras:** Funcionalidad para agregar, eliminar productos y vaciar el carrito con persistencia de datos.
* **Formulario de Contacto:** Validaciones de campos obligatorios y formato de correo electrónico antes del envío.
* **Accesibilidad:** Uso de etiquetas semánticas y labels vinculados en formularios.
